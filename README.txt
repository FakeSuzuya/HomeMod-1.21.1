Homemod

Homemod est un mod pour Minecraft conçu pour enrichir ton expérience de jeu avec des fonctionnalités innovantes et personnalisables. Développé pour Minecraft Forge, ce mod introduit des mécaniques uniques et des améliorations de gameplay qui transformeront ta manière de jouer.
Fonctionnalités

    Nouveaux Blocs et Objets : Découvre une variété de nouveaux blocs et objets qui ajoutent de la profondeur et de la diversité à ton monde Minecraft.
    Améliorations de Gameplay : Profite de nouvelles mécaniques de jeu qui enrichissent les aspects du jeu de survie et de construction.
    Compatibilité : Conçu pour fonctionner avec les versions récentes de Minecraft, assurant une expérience fluide et sans accrocs.

Installation

    Télécharge le mod : [Lien vers le fichier JAR du mod]
    Place le fichier JAR dans le dossier mods de ton répertoire Minecraft.
    Lance Minecraft avec le profil Forge pour commencer à utiliser le mod.

Aucune configuration supplémentaire n'est requise. Le mod s'active automatiquement une fois installé.

Développement

Si tu souhaites contribuer au développement de Homemod, voici comment tu peux le faire :

    Clone le dépôt :

    bash

git clone https://github.com/tonnom/homemod.git

Installe les dépendances :

bash

    cd homemod
    ./gradlew setupDecompWorkspace
    ./gradlew build

    Fais tes modifications et soumets des pull requests pour partager tes améliorations.
